# 虚拟币短线量化交易 — 完整逻辑框架

> 10U→100U 短线交易系统，技术+消息+时间+情绪四维共振。

---

## 一、技术面（你的系统核心，权重 60%）

### ① 量价衰竭 — 最重要的先行指标
```
为什么第一位：成交量是钱的脚印，价可以骗人，量不骗。

卖盘衰竭（做多）：
  连续三波下跌 → 第一波量大、第二波量中、第三波量小 → 卖的人快没了
  + 总成交量萎缩 70%+ → 全场冷清，变盘在即

买盘衰竭（做空）：
  连续三波反弹 → 一波比一波量小 → 买的人快没了
  + 总成交量萎缩 → 反弹是假的

天量+长影拒绝：
  单根K线放巨量（>均量1.5倍）但价格被砸回来 → 大户在出货/吸筹
  做空：巨量+长上影 = 拉高出货
  做多：巨量+长下影 = 砸盘吸筹
```

### ② 位置 — 你在哪
```
为什么重要：同一个做空信号，在阻力位 80% 胜率，在支撑位 20% 胜率。

阻力/支撑来源（按优先级）：
  1. SMC PD Array（机构挂单区）→ 最强
  2. 前高/前低（流动性聚集）
  3. 斐波那契 0.618 / 0.786（自然回撤位）
  4. 布林带上轨/下轨（统计极值）
  5. 成交密集区（VWAP/POC）

距离规则：
  ≤0.3% → 贴脸，一旦拒绝立刻入场
  ≤0.5% → 近，等K线收盘确认
  ≤1.0% → 快到，盯着
  >1.0% → 太远，不看不追
```

### ③ K线形态 — 入场扳机
```
射击之星 / 长上影（做空）：
  上影 ≥ 实体 × 2，下影 < 上影 × 0.3
  → 多头冲了一次被砸回来 = 多无力

锤子线 / 长下影（做多）：
  下影 ≥ 实体 × 2，上影 < 下影 × 0.3
  → 空头砸了一次被拉回来 = 空无力

吞没形态：
  熊吞没（做空）：前阳后阴，阴线实体包住阳线 → 多头被吃完
  牛吞没（做多）：前阴后阳，阳线实体包住阴线

十字星（Doji）：
  实体 < 总波动 30% → 犹豫，配合位置使用
  阻力位十字星 → 可能见顶
  支撑位十字星 → 可能见底

V形反转：
  连跌 3+ 根后急拉 → V底
  连涨 3+ 根后急砸 → V顶
  → 最强的反转信号，不用等结构确认
```

### ④ 结构 — 定大方向
```
SMC 摆动趋势 + 内部趋势：
  日线 > 4H > 1H > 15m（大周期管小周期）

  15m 摆动+内部同向 → 短线顺势，高胜率
  15m 单方向       → 短线偏方向
  15m 方向不明     → 不交易

趋势模式触发：
  日+4H+1H+15m 四同向 → 趋势模式
  → SL放宽到1.2%，追踪止损吃波段，不设紧TP

逆大势：
  做多但日线/4H空 → 逆大势 → 仓位减半
  做空但日线/4H多 → 逆大势 → 仓位减半
```

### ⑤ KDJ — 极端值硬锁
```
J < -5  → 禁止做空（极端超卖，反弹随时来）
J > 105 → 禁止做多（极端超买，暴跌随时来）

这是血的教训：ETH #1 J=-9 做空被拉爆 -2.36U
J 值极端不是加分项，是强制等待信号。
等 J 回归正常区间再判断方向。
```

### ⑥ 流动性 — 大户的陷阱
```
Equal Highs/Lows 被扫：
  多个相等高点被突破 → 多头止损被扫 → 往往是大户做空前兆
  多个相等低点被跌破 → 空头止损被扫 → 往往是大户做多前兆

用法：
  扫了 Equal Highs 后跌回来 → 空头确认
  扫了 Equal Lows 后涨回来 → 多头确认
```

## 二、消息面（权重 20%，但关键时刻变 80%）

### 红色清单（重大事件前后 30 分钟不开仓）
```
FOMC 利率决议 / 鲍威尔讲话
CPI / PPI 通胀数据
非农就业数据
SEC 重大监管决定
```

### 橙色清单（事件前后 15 分钟减半仓不追单）
```
ETF 流入/流出数据
交易所上市/下架公告
大户转账（鲸鱼报警）
交易所被盗/宕机
美股开盘（21:30）/ 收盘（04:00）
```

### 绿色清单（正常交易）
```
日常盘面波动
正常技术信号
周末低量盘（止损放宽 0.2%）
```

### 消息操作原则
```
1. 数据公布前 30 分钟 → 平掉所有持仓或拉 SL 保本
2. 数据公布后 → 等 3 根 15m K 线走完再判断方向
3. 不赌数据方向，不做"我觉得CPI会低所以做多"
4. FOMC 当天只做技术面最强的信号，仓位减半
```

## 三、时间面（权重 10%）

### 时段特性
```
亚盘（08:00-15:00 BJT）：
  波动小，量平稳 → 适合挂限价单等回调
  日韩机构盘，趋势延续为主

欧盘（15:00-21:00 BJT）：
  波动开始放大 → 趋势/反转都可能有
  与美股重叠前 1 小时最活跃

美盘（21:30-04:00 BJT）：
  波动最大，量最大 → 突破/反转最容易发生
  美股开盘 30 分钟最乱 → 不做

周末：
  量萎缩到平时的 30-50%
  假突破多 → 止损放宽，不做趋势判断
  适合复盘+学习，不适合重仓
```

### 时间纪律
```
22:00 后 → 不开新仓（除非 15m 明确结构 + 量价共振）
00:00 后 → 不开仓（凌晨决策能力下降）
睡前      → 所有持仓必须带止损，不准裸仓
```

## 四、情绪面（权重 10%）

### 资金费率
```
资金费率极高正（>0.1%）→ 多头拥挤 → 回调风险
资金费率极高负（<-0.05%）→ 空头拥挤 → 反弹风险
资金费率正常（-0.03%~0.03%）→ 中性，看技术
```

### 持仓量
```
价格涨 + 持仓量涨 → 趋势健康，跟
价格涨 + 持仓量降 → 空头在平仓，不是真涨
价格跌 + 持仓量涨 → 空头在加仓，趋势延续
价格跌 + 持仓量降 → 多头在跑路，加速跌
```

### 多空比
```
多空比 > 3:1 → 太多人做多 → 警惕回调
多空比 < 1:2 → 太多人做空 → 警惕反弹
```

## 五、三维共振总决策

### 入场条件（ALL 满足）
```
[ ] ②量价有衰竭信号 — 卖盘枯竭 or 买盘枯竭 or 天量拒绝
[ ] ③位置在关键位 ±0.5%
[ ] ④K线有拒绝信号 — 长影 / 吞没 / 十字星 / V形
[ ] ⑤J值不在极端区 — J>-5 (空) or J<105 (多)
[ ] 没有红色消息事件

→ ①②③全部满足 = 全仓 (3U)
→ ①②满足 or ①③满足 = 半仓 (2U)
→ 只有① = 等
```

### 不做的条件（任一触发）
```
❌ 日线方向相反 → 不做（15m做多但日线空 = 不做多）
❌ J 值极端 → 不做
❌ 横盘无方向 → 不做
❌ 红色消息前后 30min → 不做
❌ 美盘开盘 30min → 不做
❌ 22:00 后无明确信号 → 不做
❌ 当天已做 2 笔 → 不做
```

## 六、出场规则

### 短线模式（15m 双同向未满足）
```
SL = 入场 ± 0.8%（紧）
TP = 最近结构位（≤ 1.5%）or 1.2 × SL 距离
5 根 K 没到 TP → 手动走
浮盈到 SL 一半 → SL 拉保本
```

### 趋势模式（日+4H+1H+15m 四同向）
```
SL = 入场 ± 1.2%（宽）
TP = 远端结构位
浮盈 ≥ 5U  → SL 拉到保本
浮盈 ≥ 10U → SL 跟到浮盈 50%
不设时间限制，但 15m 结构反转 → 走
```

## 七、账户管理（10U 阶段）

```
每笔保证金 = 2-3U
杠杆 = 10x
每笔风险 ≤ 0.5U（账户 5%）
日最大亏损 = 2U（账户 20%）
→ 到线就停，明天再做

目标：
  10U → 15U（50% 收益）→ 每笔提至 4U
  15U → 25U
  25U → 50U
  50U → 100U → 启动 Phase 2（1/5 仓）
```

> 不是公式，是框架。市场会变，但衰竭+位置+拒绝这个三角不变。
> 消息面决定了你能不能做，技术面决定你做什么方向做多少。
