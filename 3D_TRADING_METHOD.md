# 三维共振交易法 (3D Resonance Trading Method)

> 基于 SMC 结构 + 量价分析 + 价格行为的交叉验证系统。
> 73 笔实盘交易迭代得出。单一维度 = 盲人摸象，三个对齐 = 高胜率。

---

## 核心原则

**三个维度都指向同一方向才开单。缺一个就等。**

| 维度 | 回答的问题 | 内容 |
|------|-----------|------|
| ① 结构 | 方向对不对 | SMC 摆动趋势 + 内部趋势 |
| ② 量价 | 力度够不够 | 成交量萎缩/放大 + 买卖盘衰竭 |
| ③ 价格行为 | 位置好不好 | 斐波那契回撤 + 支撑阻力 + 蜡烛形态 |

---

## ① 结构 — SMC 双趋势

数据来源：Mobius Quant API (`kb_klines.py indicators`)

### 方向判断

| 摆动 | 内部 | 结论 |
|------|------|------|
| -1 | -1 | **强顺势空** ✅ |
| +1 | +1 | **强顺势多** ✅ |
| -1 | +1 | 矛盾 → 刮头皮 |
| +1 | -1 | 矛盾 → 刮头皮 |

### 权重
- 摆动 70%，内部 30%
- 摆动明确时内部反向不构成矛盾，优先跟摆动
- 15m 内部转向前后是入场窗口

### 入场结构（按优先级）
1. 方向一致的最近活跃 Order Block
2. 反向 Fair Value Gap
3. 最近摆动高低点

---

## ② 量价 — 买卖盘衰竭

### 买盘衰竭信号（做空）
- 连续三波反弹量一波比一波小
- 缩量 35%+ = 买盘快用完
- 天量反弹后急缩 = 假动作

### 卖盘衰竭信号（做多）
- 连续三波下跌量一波比一波小
- 缩量 35%+ = 卖盘快用完

### 关键规则
- 成交量颜色 ≠ 买卖力量。绿色 = K 线收涨，红色 = K 线收跌
- 看量柱高矮——一根比一根矮 = 衰竭
- 天量 + 双空 + 反弹缩量 80%+ → 撤 TP，只移 SL，让利润跑

---

## ③ 价格行为 — 位置确认

### 斐波那契
- 0.618 最重要——80% 回调在此止住
- 入场在 0.618 回撤 + 支撑/阻力重合处

### 支撑阻力
- SMC 溢价区/折扣区/均衡区
- 15m 强高/强低
- 前高/前低

### 蜡烛形态
- 长上影 + 天量 @ 阻力 = 拒绝信号（做空确认）
- 长下影 + 天量 @ 支撑 = 拒绝信号（做多确认）
- Doji/十字星 @ 关键位置 = 方向即将切换

## 实战流程

```
1. 拉数据 → SMC 1H/15m + 15m K线
2. ① 结构 → 摆动+内部同向？ → 矛盾就只刮头皮
3. ② 量价 → 买盘/卖盘在衰竭吗？ → 看最近 20 根 15m K 线
4. ③ 价格 → 在关键位置吗？ → 斐波那契 + 结构 + 蜡烛
5. 三个都通过 → 出方案（入场/止损/止盈/张数）
   有一个不过 → 等信号
```

---

## 交易纪律

- **挂完单关软件**。盯盘胜率 20%，不盯盘 80%
- **SL 至少 1.2 ATR**，宁宽勿紧。SL 可以宽，不能没有
- **浮盈到 TP 一半 → SL 拉到保本**
- **浮盈到 TP 70% → 看 15m**，缩量/十字星/反转手动走
- **不追单**：入场没触发就等下一波结构或直接放弃
- **有信号就做，不设单数上限**。没信号不强求

---

## 禁止事项

- ❌ 在折扣区追空 / 溢价区追多
- ❌ 反弹时开多（逆势）
- ❌ 被扫后立刻反手
- ❌ 没 SMC 结构的单不开（"感觉到底"不是入场理由）
- ❌ 出一堆选项让用户选（只出一个方案）
